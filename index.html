jsdoc-githubify-inject

Injects jsdoc styles into pages that have been jsdoc-githubified.

View the Project on GitHub thlorenz/jsdoc-githubify-inject

  • View On GitHub

Simply drag the bookmarklet below to your bookmarks bar.

jsdoc-githubify-inject bookmarklet

Now when you are on a page that has githubified jsdocs added to either a github Wiki or a github Readme, you can simply click on the bookmarklet to add styles that make it pretty.

In case you this page by accident please review wicked to learn how to add jsdoc API pages to your wiki.
